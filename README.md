# Assignment

> Simple WordPress site page with  plugins that integrates Google Sheet

## Prerequisites

- MySQL
- PHP

## Instructions

1. Clone reporsitory to your local (If using xampp, clone it to `xampp/htdocs` folder)
2. Open MySQL database, and import file from `wordpress/database/the-ceo-mag-assignment-01.sql` 
    - Replace DB config details on `wordpress/wp-config.php` 
3. Go to `localhost/assignment-01/wordpress` on your browser
